📤 Sender Flow

  1. Visit /app - Public sender application
  2. Enter receiver details (phone, name, location)
  3. Enter amount in ZAR
  4. Get fee quote with exchange rate
  5. Create transfer → Get Lightning invoice
  6. Pay invoice using Lightning wallet (mobile/desktop)
  7. Payment confirmed → Receiver gets PIN via WhatsApp
  8. Track transfer status in real-time
Key Endpoints:
POST /api/transfers/quote
POST /api/transfers
GET /api/transfers/{id}/status
POST /api/transfers/{id}/check-payment

✓ Receiver Flow

  1. Sender pays Lightning invoice
  2. Backend: Generate 4-digit PIN and send via WhatsApp
  3. Receiver enters reference number and phone on /receiver
  4. View transfer details (amount, reference, status)
  5. Enter 4-digit PIN from WhatsApp message
  6. PIN verified → Transfer marked as RECEIVER_VERIFIED
  7. Agent can now proceed with cash payout
  8. Receiver confirmspay out completion
Key Endpoints:
GET /api/receivers/transfers/{ref}/status
POST /api/receivers/verify-pin
POST /api/receivers/resend-pin

💼 Agent Flow

  1. Login at /agent with phone + password
  2. View pending transfers in dashboard
  3. Status: Payment received, waiting for receiver verification
  4. Receiver submits PIN → Receiver verified
  5. Agent verifies receiver phone in app
  6. Transfer status: RECEIVER_VERIFIED
  7. Proceed with cash payout to receiver
  8. Confirm payout completion in app
  9. Earn commission sats on settlement
Key Endpoints:
POST /api/agent/auth/login
GET /api/agent/balance
GET /api/agent/transfers
POST /api/agent/transfers/{id}/verify

⚙️ Admin Flow

  1. Login at /admin with admin credentials
  2. Dashboard: View daily/weekly/monthly volumes
  3. Agents section: Create agents, manage balances
  4. Transfers section: Monitor all transfers in real-time
  5. Settlements section: View pending payouts
  6. Analytics: Generate reports on volumes and fees
  7. Advanced: Manager agent cash balances and disbursements
Key Endpoints:
POST /api/admin/agents
PUT /api/admin/agents/{id}/balance
GET /api/admin/transfers
POST /api/admin/agents/{id}/advance

🔄 Transfer State Machine

Every transfer progresses through these states:

State Description What Happens
INITIATED Transfer created Waiting for sender to generate invoice
INVOICE_GENERATED LND invoice created Sender ready to pay via Lightning
PAYMENT_LOCKED Payment received PIN sent to receiver, awaiting verification
RECEIVER_VERIFIED Receiver verified with PIN Ready for agent payout
PAYOUT_EXECUTED Agent executing cash payout Receiver collecting cash
SETTLED Transfer complete Ready for settlement batch
FINAL Archive complete Settlement confirmed, commissions paid
